# Community & Research Links

Use these hubs to find real caregiver stories, off-label experiments, and trial chatter. Always double-check advice with licensed clinicians.

## Reddit communities

| Community | Focus | Link |
|-----------|-------|------|
| r/cancer | Broad patient & caregiver Q&A | https://www.reddit.com/r/cancer/ |
| r/pancreaticcancer | Pancreatic + ampullary discussions, including FOLFIRINOX tricks | https://www.reddit.com/r/pancreaticcancer/ |
| r/askdocs | Anonymous questions for verified clinicians | https://www.reddit.com/r/AskDocs/ |
| r/ivermectin | Off-label ivermectin chatter (vet + human) | https://www.reddit.com/r/ivermectin/ |

Tips:
- Search within each subreddit (e.g., `site:reddit.com "ampullary" "fenbendazole"`).
- Sort by “New” for recent protocols and by “Top” for tried-and-tested tactics.

## How to contribute links back

1. Add URLs in `docs/community/` templates or open a “New Topic / Research” issue.
2. Include context: who posted, what regimen, outcome, date.
3. Tag with `#community-links` so we can keep this list fresh.

We keep this page updated as new signal sources appear. If you find a high-value Discord, Substack, or registry, submit it through the Community Hub.
